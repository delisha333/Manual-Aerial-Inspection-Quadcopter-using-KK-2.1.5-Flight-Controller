# Stability Observations

During testing, I observed that drone stability is highly sensitive to setup and calibration.

## Initial behavior

Before proper calibration:
- Drone was drifting
- Motors were not balanced
- Flight was unstable

## After calibration

After recalibrating sensors and ESC:
- Hover became stable
- Response to control improved
- Less drift observed

## Key observations

- Small parameter changes affect stability significantly
- Motor imbalance leads to tilt
- Proper ESC calibration ensures equal thrust

## Learning

Stability is not automatic.  
It depends on:
- Sensor accuracy
- Proper tuning
- Balanced hardware setup
