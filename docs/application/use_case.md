# Use Case: Aerial Inspection

This drone can be used for basic aerial inspection tasks.

## Example applications

- Inspecting rooftops
- Observing small agricultural fields
- Checking hard-to-reach areas

## Why drone is useful

Manual inspection:
- Limited visibility
- Time-consuming

Drone inspection:
- Provides top view
- Covers area quickly
- Improves observation

## Limitations

- Manual control required
- No autonomous navigation
- Limited range

---

## Conclusion

Even a simple stabilized drone can provide practical value in inspection tasks..
